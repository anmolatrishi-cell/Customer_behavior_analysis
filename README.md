This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The goal is to analyze customer shopping behavior, clean and transform data, perform structured analysis, and present insights through an interactive dashboard.

📁 Dataset

The dataset contains customer transaction and shopping behavior data.

Format: CSV
Content: Customer details, purchase information, product categories, and transactions
Objective: Identify trends and patterns in customer purchasing behavior

🛠️ Tools & Technologies
Python (Pandas) – Data loading, cleaning, and preprocessing
MySQL / SQL Server – Data querying and analysis
Power BI – Dashboard creation and visualization
Jupyter Notebook – Development environment

⚙️ Project Workflow
1. Data Loading
Imported dataset using Pandas
Checked data structure and column types
2. Exploratory Data Analysis (EDA)
Explored customer purchase patterns
Identified trends and inconsistencies
3. Data Cleaning
Handled missing values
Removed duplicates
Standardized formats for analysis
4. SQL Analysis
Loaded cleaned data into MySQL / SQL Server
Performed queries to:
Analyze total sales
Identify top customers and categories
Extract business insights
5. Dashboard Development
Built an interactive Power BI dashboard
Visualized KPIs, trends, and customer insights

📈 Results & Key Insights
📌 Top 3 categories contributed ~60–70% of total sales
📌 Identified peak shopping periods, improving timing strategies
📌 Customer segmentation revealed high-value customer groups
📌 Improved data quality by handling 100% missing/duplicate records
📌 SQL queries enabled faster reporting and structured insights
